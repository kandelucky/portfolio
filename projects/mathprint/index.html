MathPrint

Math worksheet generator for children — printable from any device

Actively in development
Unity C# Android EdTech

About

An interactive math learning system for children. The core idea: automatically generate exercises and print them from any device — phone, tablet, or computer.

Currently working on: Print scene — generating formatted worksheets ready for printing directly from the device.

Core Components

Generation Engine

Creates math exercises (addition, subtraction, etc.) with configurable difficulty, result ranges, and negative numbers. Logic ensures correctness and balance.

Printable Output

Generated tasks are ready to print from any device. Goal: instant worksheets for teachers and parents.

Android App

Simple interface for generating exercises, choosing parameters, and getting instant results. Future expansion to other platforms.

Configurable Parameters

Operation types, number ranges, 10–40 questions, multi-operand (2–5 numbers), missing operands (3 + ? = 8), comparison (>, <, =)

Key Ideas

  • Save time for teachers and parents
  • Adaptive, dynamic exercises tailored to each child
  • Print-first thinking — seamless transition from digital to physical

What Makes It Different

  • Not just an app — it's a generator + printable system
  • Focus on learning on paper, not on screen
  • Flexible logic — no fixed exercises, everything is dynamic
  • Result range validation to control output difficulty
  • Unique question generation (no duplicates)
