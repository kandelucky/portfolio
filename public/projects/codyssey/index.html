About

Codyssey is a creative educational project designed to help children learn geography, language, and logical thinking through interactive cipher-based puzzles.

The core of the project is a series of printable puzzle cards, where each card contains a hidden message encoded using a custom symbol-based system. Children decode the message by matching symbols to letters, developing problem-solving skills while learning interesting facts about different countries.
